# 프로젝트 개요

## 이름

조과로그

## 서비스 설명

조과로그는 사용자가 낚시를 다녀온 뒤 조과 기록을 남기고, 조과 사진과 기록을 편하게 다시 확인할 수 있게 하는 모바일 앱이다.

서비스의 핵심 방향은 단순 기록장이 아니라 조과 데이터를 기반으로 통계를 제공하는 것이다. 예를 들어 사용자는 "4물에 가장 많은 조과를 봤다", "무시 물때에는 조과가 좋지 않았다", "올해 출조 횟수는 몇 번이고 성공률은 몇 퍼센트다", "올해 최대어는 광어이고 최대 사이즈는 77cm다" 같은 정보를 확인할 수 있어야 한다.

또한 어종을 잘 모르는 사용자를 위해 조과 사진을 업로드하면 Gemini 3.1 AI를 활용해 어종을 판별하는 기능을 제공한다.

추가로 사용자가 잡은 어종을 포켓몬 도감처럼 모아볼 수 있는 도감 기능을 제공한다. 사용자는 자신이 잡은 어종의 일러스트 이미지를 보고, 아직 잡지 못한 어종은 잠긴 상태로 확인할 수 있다.

## 주요 사용자

- 낚시를 다니며 자신의 조과를 꾸준히 기록하고 싶은 사용자
- 조과 사진을 날짜, 위치, 물때, 날씨와 함께 정리하고 싶은 사용자
- 자신의 출조 성공률, 최대어, 어종별 기록, 물때별 성과를 알고 싶은 사용자
- 잡은 물고기의 어종을 사진으로 판별하고 싶은 초보 낚시인
- 자신이 잡은 어종을 도감처럼 수집하고 확인하고 싶은 사용자

## 핵심 사용자 행동

- 소셜 로그인으로 앱에 들어온다.
- 낚시 후 조과 기록을 등록한다.
- 조과 사진을 업로드한다.
- AI 어종 판별 결과를 확인하고 기록에 반영한다.
- 내 조과 기록을 목록과 상세 화면에서 다시 본다.
- 내가 잡은 어종 도감을 확인한다.
- 향후 통계 화면에서 물때, 날씨, 어종, 위치, 출조 횟수, 성공률, 최대어 데이터를 확인한다.

## 장기 방향

- 조과 등록 데이터를 기반으로 물때별, 날씨별, 어종별, 위치별 통계를 제공한다.
- 올해 출조 횟수, 성공률, 최대어, 대표 어종을 보여준다.
- 잡은 어종 도감을 확장해 어종별 상세 정보, 대표 이미지, 내가 잡은 횟수, 최대 사이즈를 보여준다.
- 커뮤니티 기능은 MVP 이후 별도 단계에서 검토한다.
- 물때/날씨 자동 연동은 MVP 이후 확장으로 본다.
