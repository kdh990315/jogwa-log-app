# 수익화 정책

어장관리는 1차 MVP 이후 광고/제휴 수익화를 고려한다.

사용자가 말하는 "구글 애드센스"는 모바일 앱에서는 일반적으로 Google AdMob을 의미한다. Google AdSense는 주로 웹사이트용이고, React Native 모바일 앱 수익화는 AdMob을 우선 검토한다.

쿠팡 수익화는 "쿠팡 애드센스"가 아니라 쿠팡 파트너스, 즉 제휴 마케팅 링크/배너 수익화로 정리한다.

## 수익화 채널

1차 고려 채널:

- Google AdMob: 앱 내 광고
- Coupang Partners: 낚시 장비, 미끼, 태클박스, 구명조끼, 쿨러 등 관련 상품 제휴 링크

추후 검토:

- 프리미엄 기능
- 광고 제거 결제
- 고급 통계 유료화
- 낚시 장비 추천 큐레이션

## Google AdMob 기준

앱 광고는 Google AdMob을 기준으로 한다.

원칙:

- 모바일 앱에는 Google AdMob을 우선 사용한다.
- AdSense 코드를 앱 화면에 임의로 삽입하지 않는다.
- 광고 클릭을 유도하는 문구를 쓰지 않는다.
- 사용자가 실수로 누르게 만드는 위치에 광고를 두지 않는다.
- 조과 등록, 위치 선택, AI 판별 같은 핵심 입력 흐름을 방해하지 않는다.
- 광고가 콘텐츠보다 많아 보이면 안 된다.
- loading, 빈 화면, 오류 화면처럼 콘텐츠 가치가 낮은 화면에 광고를 무리하게 넣지 않는다.

권장 광고 위치:

- 홈의 내 조과 목록 중간 또는 하단 배너
- 도감 목록 하단 배너
- 조과 상세 하단 배너
- 통계 화면이 생긴 뒤 통계 콘텐츠 아래 배너

비추천 광고 위치:

- 로그인 화면
- 조과 등록 step 중간
- 위치 선택 지도 화면 위
- 이미지 업로드 직후 버튼 주변
- AI 판별 결과 CTA 바로 옆
- 뒤로가기/저장/등록 버튼 근처

광고 형식 우선순위:

```text
배너 광고
  -> 네이티브 광고
  -> 보상형 광고
  -> 전면 광고
```

1차 MVP에서는 전면 광고를 넣지 않는다. 사용자가 조과를 등록하는 흐름을 끊으면 UX가 나빠지고 정책 리스크도 커진다.

## Coupang Partners 기준

쿠팡 수익화는 쿠팡 파트너스 제휴 링크 또는 배너를 기준으로 한다.

적합한 위치:

- 낚시 장비 추천 화면
- 조과 상세 하단의 관련 장비 추천
- 도감 상세의 관련 채비/장비 추천
- 추후 콘텐츠/가이드 화면

주의:

- 쿠팡 파트너스 링크/배너는 광고 또는 제휴 링크임을 명확히 표시한다.
- 사용자가 광고와 앱 기능을 혼동하지 않게 한다.
- 조과 등록을 완료하려면 상품 클릭이 필요한 것처럼 보이게 만들지 않는다.
- 쿠팡 파트너스 API key나 secret key를 앱 코드에 넣지 않는다.
- 쿠팡 파트너스 API를 쓰는 경우 서버 또는 Supabase Edge Function 같은 안전한 환경에서 호출한다.
- 쿠팡 정책 문구와 표시 기준은 구현 시점에 쿠팡 파트너스 공식 가이드에서 다시 확인한다.

표시 문구 예시:

```text
이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.
```

앱 화면에서는 문맥에 맞게 더 짧게 쓸 수 있지만, 정책상 필요한 고지 문구는 구현 시점에 공식 가이드로 확인한다.

## UX 기준

수익화보다 사용자의 기록 경험을 우선한다.

- 조과 등록 속도를 떨어뜨리는 광고는 넣지 않는다.
- 광고가 저장 버튼, 촬영 버튼, 위치 선택 버튼과 붙어 있으면 안 된다.
- 광고 로딩 실패가 앱 기능 실패처럼 보이면 안 된다.
- 광고 영역은 layout shift가 적도록 고정 높이를 가진다.
- 광고와 앱 콘텐츠는 시각적으로 구분한다.
- 제휴 상품 추천은 사용자의 조과/어종/낚시 유형과 관련이 있을 때만 보여준다.

## 데이터와 개인정보

- 광고 SDK 도입 시 개인정보 처리방침을 갱신한다.
- 개인화 광고를 사용할 경우 동의/거부 흐름을 검토한다.
- 위치 기반 광고나 민감 정보 기반 타겟팅은 정책 리스크가 있으므로 신중히 검토한다.
- 사용자의 조과 기록, 위치, 사진 데이터를 광고 타겟팅에 직접 사용하지 않는다.

## 구현 전 확인할 공식 문서

구현 직전에는 아래 공식 문서를 최신 기준으로 다시 확인한다.

- Google AdMob 정책
- Google Publisher Policies
- Google AdMob SDK / Expo 연동 방식
- Apple App Store Review Guidelines의 광고/추적/개인정보 관련 항목
- Google Play 정책의 광고/개인정보 관련 항목
- 쿠팡 파트너스 공식 가이드와 이용약관

## 참고한 공식 문서

- Google AdMob은 앱 개발자가 앱에 광고 공간을 만들고 수익화할 수 있는 제품이다.
- AdMob 참여자는 Google Publisher Policies와 AdMob/AdSense 프로그램 정책을 준수해야 한다.
- Google 정책은 자기 광고 클릭, 인위적 노출/클릭 유도, 오해를 부르는 광고 배치를 금지한다.
- AdSense 정책에는 AdSense code를 software application에 통합하지 말라는 제한이 있으며, 이는 AdMob과 구분된다.
- 쿠팡 파트너스는 쿠팡 상품을 링크/배너 형태로 연결하고 구매가 발생하면 수익을 얻는 제휴 마케팅 시스템이다.
